## 주피터노트북 기능

- Introspection

  - ? : 객체 정보 출력

  - ?? : 소스코드까지 출력 (가능하면)

  - **매직 명령어**

    - 앞에 % 기호를 붙여서 사용한다.
      `%magic` : 모든 매직함수의 도움말
    
      `%time <code> ` : 코드 단일 실행 후 시간 출력
    
      `%timeit <code>` : 코드를 반복 실행 후 시간출력  
    
      `%pwd` : 현재 위치
    
      `%env` : 환경 변수
      
      `%matplotlib` : 그래프를 노트북에 띄울 수 있도록 해준다(기본설정이라 따로 입력해 줄 필요는 없음)
